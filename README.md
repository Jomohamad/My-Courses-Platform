# منصتي التعليمية - My Courses Platform

Welcome to the My Courses Platform monorepo. This project is divided into two main applications:

## 1. Mobile Application (`/mobile`)
A comprehensive educational mobile app built with **Flutter**.
-   **Tech**: Flutter, Riverpod, GoRouter, Supabase.
-   **Features**: Interactive lessons, quizzes, games, and progress tracking.
-   **Instructions**: Go to `mobile/` directory and check `README.md`.

## 2. Web Application (`/web`)
A modern landing page and web platform built with **React.js**.
-   **Tech**: React 18, Vite, Premium CSS.
-   **Features**: Responsive design, course showcase, and platform features.
-   **Instructions**: Go to `web/` directory and check `README.md`.

## Quick Start

### Mobile
```bash
cd mobile
flutter pub get
flutter run
```

### Web
```bash
cd web
npm install
npm run dev
```
