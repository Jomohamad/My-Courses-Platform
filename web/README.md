# منصتي التعليمية - Web App (React)

A modern, responsive landing page and web application for the My Courses Platform, built with React and Vite.

## Features

-   **Premium Design**: Modern UI with glassmorphism, gradients, and smooth animations.
-   **Responsive Layout**: Fully responsive design for all screen sizes.
-   **Component Based**: Modular architecture using React components.

## Tech Stack

-   **Framework**: React 18
-   **Build Tool**: Vite
-   **Styling**: Vanilla CSS (with CSS Variables & Utility classes)
-   **Icons**: Lucide React

## Project Structure

```
web/
├── public/          # Static assets
├── src/
│   ├── components/  # Reusable UI components
│   ├── App.jsx      # Main application layout
│   ├── main.jsx     # Entry point
│   └── index.css    # Global styles & variables
├── index.html       # HTML entry point
├── package.json     # Dependencies & scripts
└── vite.config.js   # Vite configuration
```

## Getting Started

1.  **Navigate to web directory**:
    ```bash
    cd web
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```
